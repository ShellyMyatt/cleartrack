# ClearTrack

ClearTrack is an enterprise-style backend system for project and compliance tracking.

## Features
- Role-based access control (RBAC)
- Project and task management
- Audit logging
- Dockerized PostgreSQL database
- Flyway database migrations

## Tech Stack
- Java 17
- Spring Boot
- PostgreSQL
- Flyway
- Docker
